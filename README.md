## Quick-Reference Summary

### The 4 Disciplines:

1. **Machine Learning** -- Smart expense categorizer (Random Forest), user segmentation (K-means), financial health scoring
2. **Economics** -- Time value of money, inflation, risk-return tradeoff, behavioural economics
3. **Animation** -- Compound interest snowball, credit score building, inflation thief, risk roller coaster
4. **CSIT** -- React.js frontend, Flask backend, SQLite/PostgreSQL database, scikit-learn ML

### ML Models Used:
- Random Forest Classifier (expense categorization)
- K-Means Clustering (user segmentation)
- Linear Regression (spending trend prediction)

### Animated Modules:
- Compound Interest -- The Snowball Effect
- Credit Score -- Building Blocks
- Inflation -- The Silent Thief
- Investment Risk -- The Roller Coaster
- Budgeting -- The Bucket System

### Tech Stack:
- Frontend: React.js, HTML5, CSS3, Chart.js
- Backend: Python (Flask)
- ML: scikit-learn, Pandas, NumPy
- Database: SQLite (prototype)
